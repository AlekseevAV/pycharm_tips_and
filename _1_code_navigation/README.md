# Code navigation

## Search objects

|Search type|Shortcut            |
|-----------|--------------------|
|All        |**⇧Shift x 2**      |
|Classes    |**⌘CMD+O**          |
|Files      |**⇧Shift+⌘CMD+O**   |
|Symbols    |**⌥Options+⌘CMD+O** |
|Actions    |**⇧Shift+⌘CMD+A**   |
