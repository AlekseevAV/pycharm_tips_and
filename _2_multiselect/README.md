# Multiselect

|Action                                |Shortcut                     |
|--------------------------------------|-----------------------------|
|Select next                           |**⌃Ctrl+G**                  |
|Set multicursor in different places   |**⌥Options+<click>**         |
|Set multicursor by drag               |**⇧Shift+⌥Options+<drag>**   |
